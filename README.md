# Java DSA Interactive Mastery

This repository is designed to help you master the **basic syntax of Java for Data Structures and Algorithms (DSA)**. 

The recommended learning path is:
1. **Learn the Syntax**: Review the core concepts and patterns in [Java-syntax.md](./Java-syntax.md).
2. **Test Your Knowledge**: Once you've covered the basics, use the **Java DSA Interactive Mastery** tool to validate your skills.

## ✨ Features

- **9 Core Pillars**: Covers everything from basic Data Types to advanced Recursion and Method Signatures.
- **Search Functionality**: Quickly find syntax patterns (e.g., "sort", "heap", "map") across all pillars.
- **Dual View Modes**: Switch between **Step-by-Step** learning and a comprehensive **Cheat Sheet** view.
- **Interactive Quizzes**: Real-time feedback to reinforce learning after every section.
- **Confetti Celebration**: Visual rewards upon completing all pillars.
- **Modern UI/UX**: Premium dark-themed design with Glassmorphism and smooth animations.
- **Progress Persistence**: Automatically saves your progress using `localStorage`.
- **Code Snippets**: Optimized code snippets for common DSA patterns with "Copy to Clipboard" support.

## 📚 Pillars Included

1. **Data Types**: Primitives and char arithmetic tricks.
2. **Arrays**: Length access, static utilities, and matrix handling.
3. **Strings**: Immutability, `StringBuilder`, and `.equals()`.
4. **Lists & Maps**: `ArrayList` and `HashMap` (Frequency counting).
5. **Stacks & Queues**: `ArrayDeque` and `PriorityQueue` (Heaps).
6. **Sorting & Custom**: Lambdas and custom comparators.
7. **Math & Bounds**: Handling overflows and bit manipulation.
8. **Recursion Basics**: Base cases and memoization patterns.
9. **Method Signatures**: Handling complex return types (Nested Lists).

## 🛠️ Technology Stack
